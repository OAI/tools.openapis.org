Adding Your Tools

Want to add your tools to this repository?
You've got two options...

  • GitHub Topics

    You publish your tool on GitHub and make it available to the community. Add a topic on GitHub and your repository will show up here. New tools are added weekly.

    Adding Topics

  • Raise an Issue

    You provide tools on a commercial basis or your repository is not available in GitHub. Raise an issue and we'll add your tool to the data we use to build this website.

    Create New Issue
